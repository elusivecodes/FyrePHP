# FyrePHP

**FyrePHP** is a free, open-source framework for *PHP*.


## Table Of Contents
- [Installation](#installation)
- [Server](#server)



## Installation

**Using Composer**

```
composer create-project fyre/php myapp
```


## Server

*FyrePHP* includes a built-in web server for development purposes.

```
./bin/fyre server -p 8000
```
